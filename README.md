# 🏢 Система расчёта зарплаты | Лабораторная работа №4

Десктопное Qt-приложение для управления заработной платой с использованием паттерна "Стратегия" для расчёта бонусов.

## ✨ Основные возможности

- **CRUD операции**: Полное управление типами работ
- **3 стратегии бонусов**: Процентные, фиксированные, за стаж  
- **Работа с файлами**: Импорт и экспорт данных
- **Два окна**: Главное окно + диалоги CRUD
- **Сортировка и валидация**: Организация данных и проверка ввода
- **Перегрузка конструкторов**: Разные сценарии работы диалогов

## 🛠 Технологии

- **Язык**: C++17
- **GUI**: Qt6 Widgets
- **Сборка**: CMake
- **Паттерны**: Стратегия (BonusStrategy)

## 📂 Архитектура

- `WorkType` - данные о типах работ
- `PayrollDepartment` - логика расчётов
- `BonusStrategy` - интерфейс стратегий бонусов
- `MainWindow` - главный интерфейс
- `AddWorkDialog`/`EditWorkDialog` - CRUD операции

## 🚀 Сборка

```bash
mkdir build && cd build
cmake ..
make
./payroll_system
```
